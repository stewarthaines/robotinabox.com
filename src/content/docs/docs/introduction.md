SEED is a browser-based editor for creating EPUB 3 books from plain text. No installation required — it runs entirely in your browser, and works offline.

## What you can make

SEED supports multiple plain text input formats including Markdown, Asciidoc, Textile, Org Mode, LaTeX, and Fountain screenplays. Each format is converted to valid EPUB 3.2 with proper structure and accessibility metadata.
